# General info
This is my version of program to find shortest path using greedy algorithm for Traveling salesman problem.
The country where I am finding shortest path through all cities is Luxembourg.
Data has 980 cities in Luxembourg with their geographical coordinates.

# Greedy algorithm
1. Choose one city randomly.
2. Generate all (n-1)! cities permutations.
3. Calculate the cost of each permutation and continue the path with the lowest cost of the permutation. 
4. Keep a map of unexplored cities.
5. After reaching the last city - return the lowest total cost.
